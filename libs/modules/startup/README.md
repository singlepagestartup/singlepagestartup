# Startup Module

## 1. Purpose of the Module

The Startup module provides content widgets specific to startup pages and marketing flows.

### It solves the following tasks:

- Stores localized widget content for startup sections.
- Provides admin UI variants for managing startup widgets.

### Typical use cases:

- Startup landing page sections.
- Marketing blocks and content cards.
