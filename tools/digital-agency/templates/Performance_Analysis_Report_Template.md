## 3. Методология расчётов KPI

Определения и формулы (заполняются на основе проекта):

- ROMI = (Доход, атрибутированный маркетингу − Расходы на маркетинг) / Расходы на маркетинг
- CAC = Расходы на маркетинг / Число новых клиентов
- LTV = Средний чек × Маржинальность × Среднее число покупок за период (или модель удержания)
- Retention Rate = Клиенты активные / Клиенты в когорте периода
- Conversion Rate = Конверсии / Сессии (или клики) — указать выбранную базу
- CTR = Clicks / Impressions
- CPL = Spend / Leads

Промпт для ИИ:
`Укажи используемые формулы, источники данных для каждого слагаемого, период и единицы измерения. Если в проекте приняты иные определения — опиши их и причину.`
