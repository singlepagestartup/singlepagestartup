### Инструкция по созданию задач

**Для КАЖДОЙ связки** из `marketing_strategy.md` создай задачи по этапам воронки:

1. **Охват** — контент для канала охвата (из колонки "Охват" связки)
2. **Слежение** — контент для прогрева (для канала из колонки "Слежение")
3. **Продажа** — материалы для конверсии (для механизма из колонки "Продажа")

**Формат задачи AI:**

