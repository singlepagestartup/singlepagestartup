#### 4. Логика коммуникации (воронка продаж)

**[ОБЯЗАТЕЛЬНО]** Определить последовательность коммуникации для каждого этапа воронки.

| Этап          | Цель                                          | Ключевой посыл                                                                                          | Формат / Канал                                                                |
| ------------- | --------------------------------------------- | ------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| **Awareness** | Привлечь внимание                             | [Например: "Теряешь 15% прибыли из-за ошибок в логистике? Автоматизация решает это за 1 день"]          | [Например: Социальные сети (Meta, LinkedIn), Контекстная реклама, SEO-статьи] |
| **Interest**  | Вызвать интерес / доверие                     | [Например: "Как компании среднего бизнеса снижают затраты на логистику на 30% с помощью автоматизации"] | [Например: Лендинги, Email-рассылки, Вебинары, Кейсы]                         |
| **Desire**    | Сформировать желание / аргументировать выгоды | [Например: "Освободи 6 часов в день, повысь точность доставок до 98%, увеличь прибыль на 25%"]          | [Например: Детальные презентации, Демо-доступ, Персональные консультации]     |
| **Action**    | Призвать к действию / покупке                 | [Например: "Начни бесплатный демо-доступ на 14 дней. Без карты, без обязательств"]                      | [Например: Формы на лендингах, CTA на сайте, Email с предложением]            |

**Инструкция:**

- **Этап:** Awareness (привлечение внимания), Interest (вызов интереса), Desire (формирование желания), Action (призыв к действию)
- **Цель:** Конкретное поведение, которое мы хотим вызвать у пользователя на этом этапе
- **Ключевой посыл:** Точная формулировка сообщения, которое будет использоваться в контенте
- **Формат / Канал:** Конкретные каналы коммуникации и форматы контента (текстовые, визуальные, видео)

**Для ИИ-генерации и дизайнеров:**

- Для каждого этапа указать визуальные характеристики контента: стиль, эмоциональный акцент, композиция, цветовая палитра
- Формулировки должны быть готовы для использования в промптах генерации
- Этап воронки определяет визуальную иерархию (Awareness — яркие элементы, Action — чёткие CTA)

#### 7. Критерии успеха

**[ОБЯЗАТЕЛЬНО]** Определить измеримые метрики для оценки эффективности креативной реализации.

| Метрика          | Описание           | Источник             | Целевое значение                                 |
| ---------------- | ------------------ | -------------------- | ------------------------------------------------ |
| **ROMI**         | Возврат инвестиций | Аналитика            | [Например: 350%]                                 |
| **CTR**          | Вовлечённость      | Performance          | [Например: 3.5% для Meta Ads, 2.0% для LinkedIn] |
| **CVR**          | Конверсия          | Воронка продаж       | [Например: 2.5% для лендингов, 5% для email]     |
| **Brand Recall** | Запоминаемость     | Опрос / исследования | [Например: 40% после первого контакта]           |

**Инструкция:**

- **ROMI:** Целевое значение из **Marketing_Strategy.md** (раздел Digital Strategy)
- **CTR:** Ожидаемый показатель кликабельности для разных каналов (Meta Ads, LinkedIn, Email, сайт)
- **CVR:** Ожидаемая конверсия для разных этапов воронки (лендинги, формы, email)
- **Brand Recall:** Ожидаемая запоминаемость бренда после первого контакта (может быть измерена через опросы или аналитику вовлечённости)

**Для ИИ и дизайнеров:** Критерии успеха помогают оптимизировать креативные решения для достижения целевых метрик.
