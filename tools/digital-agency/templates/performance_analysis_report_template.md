## 2. Методология показателей

- **Возврат инвестиций** = (Доход − РасходМаркетинга) / РасходМаркетинга
- **Стоимость привлечения** = РасходМаркетинга / НовыеКлиенты
- **Жизненная ценность** = СреднийЧек × Частота × СрокУдержания
- **Удержание** = КлиентыАктивные / КлиентыКогорты
- **Конверсия** = Конверсии / Сессии
- **Кликабельность** = Клики / Показы
- **Стоимость лида** = Расход / Лиды
