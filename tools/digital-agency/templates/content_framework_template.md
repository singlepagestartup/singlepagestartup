# Архитектура контента

## 1. Принципы создания текстов

**Подход на основе данных:**

- Каждый текст работает на конкретный показатель в порядке приоритета: **сначала масштаб** (охваты, просмотры, вовлечённость, подписки, клики), **затем конверсия при достаточном объеме данных** (увеличение возврата инвестиций, снижение стоимости привлечения, увеличение ценности, увеличение удержания)
- Конкретные цифры вместо общих фраз ("экономия 30%" вместо "значительная")

**Психологические принципы:**

- Социальное доказательство (кейсы, отзывы, статистика)
- Дефицит (ограничения по времени/количеству)
- Авторитет (экспертность, партнёрства)
- Избегание потерь (акцент на потерях от бездействия)
- Выгода против функции (фокус на выгодах)
- Якорение (конкретные цифры как якоря)

**Структура связки (контентная роль каждого шага):**

- **Охват:** Привлечь внимание → показать экспертность → дать причину подписаться
- **Слежение:** Дать ценность → построить доверие → подвести к офферу
- **Продажа:** Снять возражения → показать результат → призвать к действию

**Воронка внутри каждого шага:**

- **Проблема + любопытство** → **Решение + доверие** → **Выгоды + призыв к действию**

### Общие принципы (для всех платформ):

#### Воронка Охват → Слежение → Продажа

**Охват:**

- Цель: Привлечь внимание → показать экспертность → дать причину подписаться
- Формат: Короткие видео, статьи, посты
- CTA: Переход на канал слежения (Telegram, Email, VK-сообщество)

**Слежение:**

- Цель: Дать ценность → построить доверие → подвести к офферу
- Форматы: Telegram-канал/бот, Email-рассылка, VK-сообщество, WhatsApp-канал
- CTA: Ссылка на продажу, регистрация на вебинар, запись на консультацию

**Продажа:**

- Цель: Снять возражения → показать результат → призвать к действию
- Формат: Консультация, лендинг, вебинар, чат-бот
- CTA: Оплата, запись, оставить заявку

## 4. Формулы генерации

### Универсальная формула заголовка

```
[Проблема клиента] + [Конкретная цифра] + [Вопрос/Решение] = Заголовок
```

Примеры:

- "Как увеличить продажи на 40% за 30 дней"
- "5 ошибок, которые убивают вашу конверсию"
- "Почему 90% стартапов провалились с рекламой"

### Формула поста для соцсетей

```
[Крючок] + [Проблема] + [Решение] + [CTA] = Пост
```

### Формула статьи

```
[Заголовок с результатом] + [Введение с обещанием] + [Проблема] + [Решение пошагово] + [Кейс] + [CTA] = Статья
```

### Формула email

```
[Тема с интригой] + [Персонализация] + [Польза] + [CTA] + [P.S.] = Письмо
```

### Формула видео

```
[Крючок 0-3 сек] + [Проблема 3-8 сек] + [Решение 8-40 сек] + [CTA 40-60 сек] = Видео
```

## 5. Лидмагниты

### Типы лидмагнитов

| Тип         | Формат       | Объём      | Для кого              | Время создания |
| ----------- | ------------ | ---------- | --------------------- | -------------- |
| Чек-лист    | PDF          | 1-3 стр    | Занятые               | 2-4 часа       |
| Гайд        | PDF          | 10-20 стр  | Хотят глубже          | 1-2 дня        |
| Шаблон      | Notion/Excel | —          | Практики              | 4-8 часов      |
| Мини-курс   | Email/видео  | 5-7 уроков | Учащиеся              | 3-5 дней       |
| Вебинар     | Видео        | 30-60 мин  | Хотят видеть эксперта | 1 день         |
| Калькулятор | Веб/Excel    | —          | Хотят посчитать       | 1-3 дня        |

### Структура PDF-лидмагнита

```
1. Обложка (название + визуал)
2. О чём материал (1 абзац)
3. Для кого (буллеты)
4. Основной контент
5. Бонус
6. CTA (следующий шаг)
7. Контакты
```

## 6. Офферы и продающие элементы

### Структура оффера

```
[Что получает] + [За какой срок] + [С какой гарантией] + [По какой цене] + [Ограничение]
```

### Типы офферов

| Этап      | Тип                    | Пример                         |
| --------- | ---------------------- | ------------------------------ |
| Холодный  | Бесплатная диагностика | «Бесплатный аудит за 30 минут» |
| Тёплый    | Пробный период         | «Первый месяц за 1₽»           |
| Горячий   | Основной продукт       | «Полный курс + бонусы»         |
| Повторный | Апсейл                 | «Модуль со скидкой 30%»        |

## 7. Скрипты продаж

### Структура консультации

| Этап         | Время     | Цель             | Фразы                          |
| ------------ | --------- | ---------------- | ------------------------------ |
| Раппорт      | 2-3 мин   | Контакт          | «Расскажите о себе...»         |
| Квалификация | 5 мин     | Понять клиента   | «Какая ситуация?»              |
| Диагностика  | 10-15 мин | Выявить боли     | «Что мешает?»                  |
| Презентация  | 10 мин    | Показать решение | «Вот как решаем...»            |
| Возражения   | 5-10 мин  | Снять сомнения   | «Понимаю, давайте разберём...» |
| Закрытие     | 3-5 мин   | Решение          | «Что нужно, чтобы начать?»     |

### Работа с возражениями

| Возражение              | Ответ                                  |
| ----------------------- | -------------------------------------- |
| «Дорого»                | Сравнить с потерями / разбить на части |
| «Надо подумать»         | Что именно смущает?                    |
| «У конкурентов дешевле» | Показать разницу в ценности            |
| «Не сейчас»             | Что изменится потом?                   |

## Инструкция для ИИ-генератора контента

При создании контента для проекта:

1. **Прочитай соответствующую платформу:** `platforms/[платформа].md` — там технические требования (форматы, разрешения, длительность)
2. **Следуй правилам из этого файла:**
   - Принципы создания текстов (раздел 1)
   - Стиль коммуникации (раздел 3)
   - Формулы генерации (раздел 4)
3. **Используй артефакты проекта:**
   - `project/client_brief.md` — ЦА, продукт, боли
   - `project/marketing_strategy.md` — офферы, темы
   - `project/project_plan.md` — конкретные задачи
4. **Проверь качество:**
   - Наличие крючка, проблемы, решения, CTA
   - Конкретные цифры вместо общих фраз
   - Социальное доказательство где применимо
   - Соответствие стилю коммуникации проекта
