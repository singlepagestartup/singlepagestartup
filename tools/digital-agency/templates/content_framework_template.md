# Архитектура контента

## 1. Принципы создания текстов

**Подход на основе данных:**

- Каждый текст работает на конкретный показатель: увеличение возврат инвестиций, снижение стоимости привлечения, увеличение ценности, увеличение конверсии, увеличение удержания
- Конкретные цифры вместо общих фраз ("экономия 30%" вместо "значительная")

**Психологические принципы:**

- Социальное доказательство (кейсы, отзывы, статистика)
- Дефицит (ограничения по времени/количеству)
- Авторитет (экспертность, партнёрства)
- Избегание потерь (акцент на потерях от бездействия)
- Выгода против функции (фокус на выгодах)
- Якорение (конкретные цифры как якоря)

**Структура связки (контентная роль каждого шага):**

- **Охват:** Привлечь внимание → показать экспертность → дать причину подписаться
- **Слежение:** Дать ценность → построить доверие → подвести к офферу
- **Продажа:** Снять возражения → показать результат → призвать к действию

**Воронка внутри каждого шага:**

- **Проблема + любопытство** → **Решение + доверие** → **Выгоды + призыв к действию**

### 2.1. Короткие вертикальные видео

#### 2.1.1. VK Клипы

**Роль в связке:** Охват → переход в VK-сообщество или Telegram-канал

**Технические требования:**

- Формат: 9:16 (вертикальное)
- Разрешение: 1080×1920 px
- Длительность: 15-60 секунд (оптимально 30-45)
- Размер файла: до 2 ГБ
- Субтитры: ОБЯЗАТЕЛЬНЫ (80% смотрят без звука)
- Музыка: только без авторских прав (библиотека VK)

**Структура ролика (по секундам):**

| Секунды | Элемент           | Что показывать                            | Текст на экране                         |
| ------- | ----------------- | ----------------------------------------- | --------------------------------------- |
| 0-3     | Крючок            | Лицо крупно + движение / неожиданный кадр | Провокационный вопрос или факт          |
| 3-8     | Проблема          | Визуализация боли ЦА                      | Усиление проблемы                       |
| 8-20    | Решение (часть 1) | Демонстрация первого шага                 | Польза #1                               |
| 20-35   | Решение (часть 2) | Демонстрация результата                   | Польза #2 + доказательство              |
| 35-45   | CTA               | Указание на ссылку/профиль                | «Подробнее в описании» / «Подписывайся» |

**Типы крючков (первые 3 секунды):**

1. Провокационный вопрос: «Почему 90% бизнесов теряют деньги на рекламе?»
2. Шокирующий факт: «Я потерял 500 000₽, пока не понял это...»
3. Прямое обращение: «Если ты [описание ЦА], смотри до конца»
4. Обещание: «За 45 секунд покажу, как [результат]»
5. Интрига: «Это изменило мой бизнес навсегда»

**Требования к визуалу:**

- Первый кадр: лицо крупным планом или яркий объект
- Смена кадров: каждые 2-4 секунды
- Текст на экране: крупный, читаемый, контрастный фон
- Указатели: стрелки, подчёркивания ключевых слов
- Финальный кадр: визуальный CTA (стрелка на профиль)

**CTA варианты:**

- «Подписывайся, чтобы не пропустить продолжение»
- «В закреплённом посте — полный гайд»
- «Ссылка в описании — бесплатный чек-лист»
- «Переходи в Telegram — там ещё 10 таких советов»

**Хештеги:** 3-5 релевантных, включая #vkклипы #[ниша] #[тема]

**Метрики успеха:**

- Досмотры > 30%
- Переходы в профиль > 2%
- Подписки/переходы по ссылке > 0.5%

#### 2.1.2. YouTube Shorts

**Роль в связке:** Охват → переход на YouTube-канал или внешний ресурс

**Технические требования:**

- Формат: 9:16 (вертикальное)
- Разрешение: 1080×1920 px
- Длительность: 15-60 секунд (оптимально 30-50)
- Субтитры: ОБЯЗАТЕЛЬНЫ
- Музыка: библиотека YouTube или без АП

**Структура ролика:**

| Секунды | Элемент          | Цель               | Примечания                         |
| ------- | ---------------- | ------------------ | ---------------------------------- |
| 0-1     | Супер-крючок     | Остановить скролл  | Максимально сильный визуал + текст |
| 1-5     | Контекст         | Объяснить, о чём   | Быстро, без воды                   |
| 5-25    | Основной контент | Дать ценность      | Пошагово, с визуалом               |
| 25-40   | Доказательство   | Показать результат | Цифры, до/после                    |
| 40-50   | CTA              | Направить дальше   | Подписка / полное видео            |

**Особенности YouTube Shorts:**

- Алгоритм ценит удержание — первые 3 секунды критичны
- Комментарии влияют на продвижение — задавать вопросы
- Loops (петли) — видео начинается с конца предыдущего просмотра
- Связка с длинными видео — «Полная версия на канале»

**Типы контента с высоким охватом:**

1. Разборы ошибок: «3 ошибки, которые убивают ваш [X]»
2. Пошаговые инструкции: «Сделай это за 30 секунд»
3. До/После: трансформации, результаты
4. Реакции: на тренды, новости ниши
5. Мини-туториалы: одно конкретное действие

**CTA варианты:**

- «Подпишись — новый совет каждый день»
- «Полное видео по ссылке в описании»
- «Напиши в комментариях, если хочешь продолжение»
- «Сохрани, чтобы не потерять»

**Метрики успеха:**

- Просмотры: зависит от ниши
- Удержание > 40%
- Подписки с Shorts > 1% от просмотров

**Роль в связке:** Охват → переход в профиль Instagram / Telegram

**Технические требования:**

- Формат: 9:16 (вертикальное)
- Разрешение: 1080×1920 px
- Длительность: 15-90 секунд (оптимально 15-30)

| ------- | -------- | ---------------------------------- |

| 5-20    | Контент  | Динамично, с трендовыми переходами |
| 20-25   | CTA      | Переход в профиль / ссылка в bio   |

- Трендовые звуки: использовать звуки с растущей популярностью

**Связка с профилем:**

- Ссылка в bio → на лидмагнит

- «Сохрани, чтобы не потерять»

- «Напиши + в директ — пришлю чек-лист»
- «Подпишись, чтобы видеть такой контент»

**Метрики успеха:**

- Охват > подписчиков аккаунта
- Сохранения > 3%
- Переходы в профиль > 2%

### 2.2. Длинные видео

#### 2.2.1. YouTube (длинные видео)

**Роль в связке:** Охват + прогрев → подписка на канал / переход на внешний ресурс

**Технические требования:**

- Формат: 16:9 (горизонтальное)
- Разрешение: 1920×1080 (Full HD) или 4K
- Длительность: 8-20 минут (оптимально 10-15)
- Превью: 1280×720 px, лицо + текст + контраст

**Структура видео:**

| Минуты      | Элемент     | Содержание                           | Цель                   |
| ----------- | ----------- | ------------------------------------ | ---------------------- |
| 0:00-0:30   | Крючок      | Обещание результата + интрига        | Удержать первые 30 сек |
| 0:30-2:00   | Введение    | Контекст + представление + структура | Объяснить, о чём видео |
| 2:00-4:00   | Проблема    | Боли ЦА с примерами                  | Создать резонанс       |
| 4:00-12:00  | Решение     | Пошаговый разбор с примерами         | Дать ценность          |
| 12:00-14:00 | Кейс        | Реальный пример применения           | Доказательство         |
| 14:00-15:00 | Итоги + CTA | Резюме + призыв к действию           | Конверсия              |

**Элементы удержания:**

- Pattern interrupts каждые 2-3 минуты (смена кадра, B-roll, графика)
- Open loops: «Об этом расскажу через 2 минуты»
- Визуальные подсказки: стрелки, выделения
- Глава (timestamps) для навигации
- Карточки YouTube с ссылками

**Превью (thumbnail):**

- Лицо с эмоцией (удивление, радость)
- 3-5 слов крупным текстом
- Контрастные цвета
- Читаемость на мобильных (маленький размер)

**CTA в видео:**

- Середина: «Подпишитесь, если полезно»
- Конец: «Ссылка в описании — бесплатный [лидмагнит]»
- Описание: ссылки на ресурсы, Telegram, сайт

**Метрики успеха:**

- Удержание > 50%
- CTR превью > 5%
- Подписки с видео > 2%

### 2.3. Статьи и публикации

#### 2.3.1. Яндекс Дзен (статьи)

**Роль в связке:** Охват через рекомендации → подписка на канал Дзен / внешний ресурс

**Технические требования:**

- Длина: 2000-5000 знаков (оптимально 3000-4000)
- Изображения: минимум 2-3, горизонтальные
- Обложка: 1200×630 px, яркая, с текстом

**Структура статьи:**

```
## Заголовок (50-70 символов)
[Цепляющий, с выгодой или интригой, без кликбейта]

### Введение (200-300 знаков)
[Проблема + обещание решения + для кого статья]

### Основная часть
#### Подзаголовок 1 (проблема или первый шаг)
[Абзац 300-500 знаков + изображение]

#### Подзаголовок 2 (решение или второй шаг)
[Абзац 300-500 знаков + изображение]

#### Подзаголовок 3 (пример или третий шаг)
[Абзац 300-500 знаков + пример/кейс]

### Заключение + CTA (200-300 знаков)
[Резюме + ссылка на лидмагнит / канал Дзен / Telegram]
```

**Особенности Яндекс Дзен:**

- Дочитываемость > 40% — главная метрика
- Алгоритм продвигает статьи с высоким вовлечением
- Ссылки на внешние ресурсы разрешены

- Показывается в публичных каналах с 1000+ подписчиков

- Нативное размещение с меткой «Реклама»
- Нет кликабельных ссылок в тексте — только кнопка
- Модерация: проверяется текст, медиа и целевой канал/бот
- Таргетинг: тематика канала, язык, страна

**Ограничения:**

- Запрещён шок-контент
- Нельзя использовать логотипы без прав на бренд
- Нельзя использовать личные данные
- Нарушение прав запрещено

```

[Крючок — боль или выгода] + [Решение] + [Кнопка CTA]
```

**Текстовое объявление (для канала):**

```

Хотите зарабатывать на крипте без риска?
Разбираем стратегии в закрытом канале.
3000+ подписчиков уже с нами
[Кнопка: Подписаться]

```

**Текст + изображение (для продукта):**

```

[Баннер 16:9: скриншот интерфейса / результат]

Устали от хаоса в задачах?
Наш планировщик наводит порядок за 5 минут.
[Кнопка: Попробовать бесплатно]

```

**Текст + видео (для услуг):**

```

[Видео 3-15 сек: демонстрация результата / отзыв]

Английский за 3 месяца — реально.
Бесплатный урок с носителем.
Запишитесь → [ссылка]

```

**Советы:**

- Первые слова — самые важные
- Конкретика вместо общих фраз
- Цифры работают лучше слов
- CTA в конце обязателен
- Тестировать разные варианты текста
