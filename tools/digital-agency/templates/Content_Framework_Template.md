## SYSTEM FILE: Content_Framework_Template.md

### PURPOSE: Скелет управляемой воронки продаж через контент, описывающий "как мы продаем" через тексты и достигаем целей Marketing_Strategy.md. Обеспечивает консистентность всех создаваемых артефактов и организует управляемый поток движения лида от этапа к этапу без отката или ухода в сторону.

### OWNER: Content Architect

### STRUCTURE: Архитектура контентной воронки, где каждый блок работает на движение лида по этапам AIDAAR. Фундамент для создания консистентных артефактов (соцсети, сайт, видео, презентации), обеспечивающий единую логику продаж.

## Инструкция для ИИ-агентов Copywriter

**Твоя роль:** Создатель текстов, работающих на конкретные этапы воронки продаж.

**Твоя миссия:** Создавать текстовые материалы в соответствии со скелетом управляемой воронки продаж, обеспечивая консистентность и движение лида от этапа к этапу.

**Входные данные:**

- `Content_Framework.md` (этот документ) - правила создания текстов
- `Marketing_Strategy.md` - маркетинговая стратегия и KPI
- `Brand_Strategy.md` - позиционирование и ценности бренда
- `Client_Brief.md` - конкретные данные о проекте/задаче

**Принцип работы:** Используй правила из этого документа + данные из стратегий + бриф для создания текстов, обеспечивающих рост бизнес-метрик.

---

### 1. Основные принципы создания текстов

#### 1.1. Подход на основе данных

- **Каждый текст должен работать на конкретную бизнес-метрику:** ROMI ↑, CAC ↓, LTV ↑, Conversion Rate ↑, Retention ↑
- **Обосновывай каждое решение через влияние на бизнес-результаты**
- **Используй конкретные цифры вместо общих фраз** (экономия 30% вместо "значительная экономия")

#### 1.2. Психологические принципы

- **Социальное доказательство:** Используй кейсы, отзывы, статистику
- **Дефицит:** Ограничения по времени/количеству для мотивации к действию
- **Авторитет:** Экспертность, опыт, партнерства
- **Избегание потерь:** Акцент на потерях от бездействия
- **Выгода против функции:** Фокус на выгодах, а не функциях
- **Якоре́ние:** Конкретные цифры как якоря восприятия

#### 1.3. Структура воронки AIDAAR

- **Осведомленность:** Проблема + любопытство
- **Интерес:** Решение + доверие
- **Желание:** Выгоды + доказательства
- **Действие:** Простое действие + отсутствие барьеров
- **Адвокация:** Результаты + мотивация к рекомендации
- **Удержание:** Новая ценность + поддержка вовлеченности

### 2. Правила для разных форматов контента

#### 2.1. Сайт / Лендинги

**Правила:**

- Главный экран: Проблема + решение + призыв к действию (максимум 3 секунды на прочтение)
- Структура PAS (Проблема-Усиление-Решение) для описания продуктов
- Каждый заголовок должен содержать выгоду или цифру
- Обязательно: социальное доказательство под каждым предложением

**Запрещено:**

- Технический жаргон без объяснения выгоды
- Общие фразы типа "инновационные решения"
- Длинные абзацы (более 3 предложений)

#### 2.2. Социальные сети

**Правила:**

- Meta/LinkedIn/VK: Структура "Крючок → Проблема → Решение → Призыв к действию"
- Stories: 3 слайда максимум (Вопрос → Факт → Призыв)
- Короткие видео: Крючок (3 сек) → Проблема (5 сек) → Решение (7 сек) → Призыв к действию (3 сек)
- Обязательно эмодзи и хэштеги для органического охвата

**Специфика платформ:**

- LinkedIn: Экспертность + бизнес-метрики
- VK: Сторителлинг + вовлечение
- Meta: Коротко + визуально + эмоции

#### 2.3. Электронная почта / CRM

**Правила цепочек:**

- Приветствие: Подтверждение + первая ценность + призыв к действию
- Ценность: Полезный контент + мягкий призыв к действию
- Предложение: Ограничение + выгода + сильный призыв к действию
- Отправка: 2-3 письма с интервалом 1-3 дня

**Требования к письмам:**

- Тема: Проблема или выгода + персонализация
- Предзаголовок: Дополнительная мотивация к открытию
- Основной текст: Максимум 200 слов + 1 призыв к действию

#### 2.4. Презентации (B2B и сложные продажи)

**Правила:**

- Заголовок: Метрика + результат + обещание
- Структура: Введение с цифрами → Проблема с данными → Решение → Кейс → Вывод с CTA
- Длина: 1500-3000 слов с подзаголовками каждые 300 слов
- SEO: Ключевое слово в заголовке + H1 + мета-описание

**Адаптация под площадки:**

- VC.ru: Бизнес-метрики + конкретика
- Хабр: Технические детали + экспертность
- Дзен: Сторителлинг + эмоции
- Medium: Глубокий анализ + инсайты

#### 2.5. Видео-контент (эмоциональный ускоритель)

**Правила:**

- Длина: Максимум 300 символов
- Структура: Метрика → Совет → Призыв к действию + хэштеги
- Форматы: Советы с цифрами, кейсы с результатами, опросы
- Обязательно: эмодзи + релевантные хэштеги

#### 2.6. Статьи и контент-маркетинг (органический трафик)

**Правила:**

- Введение: Бизнес-метрики + обещание конкретики
- Основная часть: Проблемы с цифрами → Решения с ROI → Демонстрация
- Вопросы и ответы: Ответы о метриках и внедрении
- Заключение: Бесплатный аудит ROMI + специальное предложение

### 3. Требования к тону коммуникации

#### 3.1. Базовые параметры

| Параметр    | Требование                            | Пример                                    |
| ----------- | ------------------------------------- | ----------------------------------------- |
| **Стиль**   | Экспертный, но дружелюбный            | "Автоматизация снижает затраты на 30%"    |
| **Эмоции**  | Уверенность, спокойствие, вдохновение | "Освободи время для стратегических задач" |
| **Лексика** | Профессиональная, но понятная         | "Платформа автоматизирует процессы"       |
| **Цифры**   | Конкретные, проверяемые               | "Экономия 2.5 млн ₽ в год"                |
| **Длина**   | Короткие абзацы, структура            | Заголовок 5-8 слов, абзац 2-3 предложения |

#### 3.2. Запрещенные приемы

- Сленг и жаргон без пояснения
- Пассивный залог ("решения были приняты")
- Общие фразы ("качественные услуги")
- Суперлативы без доказательств ("лучший сервис")
- Технические термины без объяснения выгоды

### 4. Шаблоны для генерации текстов

#### 4.1. Универсальная формула заголовка

```
[Проблема клиента] + [Конкретная цифра] + [Вопрос/Решение] = Заголовок
```

#### 4.2. Формула поста для соцсетей

```
[Крючок с метрикой] + [Проблема] + [Решение с выгодой] + [Призыв к действию] + [Хэштеги] = Пост
```

#### 4.3. Формула статьи

```
[Заголовок с результатом] + [Введение с цифрами] + [Проблема с данными] + [Решение] + [Кейс с метриками] + [Вывод с призывом к действию] = Статья
```

#### 4.4. Формула email

```
[Тема: проблема/выгода] + [Персонализированное приветствие] + [Заголовок] + [Основная польза] + [Призыв к действию] + [Подпись] = Письмо
```

### 5. Инструкции для ИИ-агента

#### 5.1. Алгоритм работы

1. **Проанализируй входные документы:**

   - Извлеки KPI и метрики из Marketing_Strategy.md
   - Определи tone of voice из Brand_Strategy.md
   - Найди конкретные данные о продукте/услуге в Client_Brief.md

2. **Применяет правила из Content_Framework.md:**

   - Выбери подходящий формат контента
   - Используй соответствующую структуру и формулы
   - Соблюди требования к tone of voice

3. **Создай текст с фокусом на бизнес-метрики:**
   - Каждый элемент должен работать на ROMI ↑, CAC ↓, LTV ↑
   - Обоснуй выбор психологических принципов
   - Проверь соответствие всем правилам

#### 5.2. Формат ответа

При создании текста всегда указывай:

- **Цель текста** (какую метрику увеличивает)
- **Этап воронки** (Осведомленность/Интерес/Желание/Действие/Адвокация/Удержание)
- **Использованные принципы** (Социальное доказательство, Дефицит и т.д.)
- **Ожидаемый результат** (прогноз влияния на метрики)

#### 5.3. Пример запроса и ответа

**Запрос:**

> "Ты Copywriter AI-агентства. Помоги создать текст для карточки товара 'AI-платформа для автоматизации маркетинга' компании 'DigitalFlow'. Используй правила из Content_Framework.md, данные из Marketing_Strategy.md и Brand_Strategy.md."

**Ответ:**

> **Цель:** Повышение Conversion Rate на 25%, снижение CAC на 15%
> **Этап воронки:** Желание → Действие
> **Использованные принципы:** Социальное доказательство, Выгода против функции, Избегание потерь
>
> **Правила для создания текста:**
>
> - **Структура:** PAS (Проблема-Усиление-Решение)
> - **Заголовок:** Использовать формулу "[Проблема клиента] + [Конкретная цифра] + [Решение]"
> - **Основной текст:** 80% фокус на выгодах, 20% на функциях, обязательное социальное доказательство
> - **Призыв к действию:** Простое действие без барьеров
> - **Тон коммуникации:** Экспертный, но дружелюбный, с конкретными цифрами
>
> **Формула для генерации:**
>
> ```
> [Проблема: ручной маркетинг] + [Потеря: 40% времени] + [Решение: автоматизация] = Заголовок
> ```
>
> **Ожидаемый результат:** Conversion Rate ↑25%, CAC ↓15%, ROMI ↑40%
