# Pinterest

**Общая информация:**

- **Аудитория:** Глобальная, 70% женщины, 25-45 лет, 450+ млн активных пользователей/месяц
- **Особенности:** Визуальная поисковая платформа, длинный срок жизни контента (месяцы/годы), высокий покупательский интент
- **Преимущества:** Долгосрочный трафик, SEO-эффект, высокая конверсия в покупки (47% пользователей совершили покупку)
- **Статистика:** Pinterest Analytics, интеграция с Google Analytics

## 1. Пины (Pins)

**Роль в связке:** Охват (поисковый трафик) → Продажа (переход на сайт/магазин)

**Технические требования:**

- **Формат:** JPG, PNG
- **Соотношение сторон:** 2:3 (оптимально), 1:2.1 (максимум)
- **Размер:** 1000×1500 px (рекомендуется)
- **Вес файла:** до 20 МБ
- **Текст на изображении:** допустим, улучшает кликабельность

**Структура пина:**

```
Изображение:
  - Вертикальное 2:3 (1000×1500 px)
  - Яркие контрастные цвета
  - Текст-заголовок на изображении (читаемый шрифт)
  - Продукт/результат в центре

Заголовок: до 100 символов (отображаются ~60 на превью)
Описание: до 500 символов (с ключевыми словами)
URL: ссылка на страницу сайта/товара
Alt-text: описание изображения для SEO
```

**Пример оформления пина:**

```
Заголовок: "15 идей дизайна маленькой кухни | Экономия пространства"

Описание:
Ищете идеи для маленькой кухни?
Эти 15 дизайнерских решений помогут максимально использовать пространство.
✅ Умные системы хранения
✅ Многофункциональная мебель
✅ Визуальное расширение

#дизайнкухни #маленькаякухня #интерьер #ремонт

URL: https://yoursite.com/kitchen-ideas
```

**Типы пинов:**

| Тип             | Описание                                       | Использование                  |
| --------------- | ---------------------------------------------- | ------------------------------ |
| Стандартный пин | Одно изображение + описание + ссылка           | Статьи, товары, идеи           |
| Видео-пин       | Вертикальное видео 9:16                        | Рецепты, туториалы, DIY        |
| Карусель        | До 5 изображений                               | Подборки, до/после, коллекции  |
| Idea Pin        | До 20 страниц (фото/видео), без внешней ссылки | Нативный контент для платформы |
| Product Pin     | Автосинхронизация с каталогом товаров          | E-commerce, интернет-магазины  |

## 2. Видео-пины

**Роль в связке:** Охват (высокий приоритет в алгоритме)

**Технические требования:**

- **Формат:** MP4, MOV
- **Соотношение сторон:** 9:16 (вертикальное), 1:1 (квадратное)
- **Длительность:** 4 секунды - 15 минут
- **Размер файла:** до 2 ГБ
- **Разрешение:** минимум 720p

**Структура видео:**

```
0-3 сек: Крючок (визуальный + текст на экране)
3-15 сек: Проблема или интрига
15-60 сек: Основной контент (решение, процесс)
Финал: Результат + CTA

Обязательно:
- Субтитры (многие смотрят без звука)
- Текст на экране (ключевые моменты)
- Брендинг (логотип в углу)
```

**Темы для видео-пинов:**

- Рецепты (пошаговое приготовление)
- DIY и поделки
- Уроки макияжа
- Лайфхаки
- До/После (ремонт, преображение)
- Туториалы

## 3. Idea Pins (ранее Story Pins)

**Роль в связке:** Вовлечение и рост подписчиков (без внешних ссылок)

**Технические требования:**

- До 20 страниц (изображения или видео)
- Формат: 9:16 (вертикальный)
- Размер: 1080×1920 px

**Особенности:**

- Нельзя добавить внешнюю ссылку
- Можно добавить теги тем (#тема)
- Стикеры, текст, музыка
- Приоритет в алгоритме (Pinterest продвигает нативный контент)

**Когда использовать:**

- Многошаговые инструкции
- Серии идей (например, "10 идей декора спальни")
- Истории и процессы
- Рост подписчиков (вовлечение → подписка)

## 4. Доски (Boards)

**Роль в связке:** Организация контента, SEO

**Структура доски:**

```
Название: ключевые слова (например, "Идеи дизайна кухни")
Описание: до 500 символов с ключевыми словами
Обложка: лучший пин доски
Категория: выбор из списка Pinterest
```

**Типы досок:**

- **Публичные:** видны всем, индексируются поисковиками
- **Секретные:** только для вас (для подготовки контента)
- **Групповые:** несколько авторов могут добавлять пины

**Организация досок:**

Пример для интерьерного дизайнера:

```
Доска 1: "Дизайн кухни — Идеи и вдохновение"
Доска 2: "Дизайн спальни — Современные решения"
Доска 3: "Гостиная — Уютный интерьер"
Доска 4: "Ванная комната — Стильные проекты"
Доска 5: "DIY для дома — Декор своими руками"
```

**Оптимизация досок:**

- 10-20 досок по тематике
- Каждая доска = отдельная тема/категория
- Регулярное обновление (добавление новых пинов)
- Ключевые слова в названии и описании

## 6. Контент-стратегия

**Частота публикаций:**

- Новые аккаунты: 5-10 пинов/день
- Зрелые аккаунты: 10-30 пинов/день
- Распределение: утро (8-10), обед (13-14), вечер (20-22)

**Соотношение контента:**

- 80% — чужой контент (репины качественных пинов из вашей ниши)
- 20% — свой контент (пины с вашего сайта/магазина)

**Принцип: качество > количество**

**Типы контента по целям:**

| Цель             | Тип контента                           | Пример                        |
| ---------------- | -------------------------------------- | ----------------------------- |
| Трафик на сайт   | Стандартные пины (со ссылкой)          | Статьи блога, товары          |
| Вовлечение       | Видео-пины, Idea Pins                  | Туториалы, лайфхаки           |
| Продажи          | Product Pins (каталог товаров)         | E-commerce, интернет-магазины |
| Рост подписчиков | Качественный визуал + серийный контент | Коллекции, подборки           |

## 7. Pinterest Ads (реклама)

**Роль в связке:** Охват (платный трафик) → Продажа

**Форматы рекламы:**

| Формат          | Описание                             | Использование                 |
| --------------- | ------------------------------------ | ----------------------------- |
| Standard Pins   | Обычные пины (продвигаемые)          | Трафик, конверсии             |
| Video Pins      | Видео-реклама                        | Вовлечение, узнаваемость      |
| Carousel Ads    | До 5 изображений в карусели          | Подборки, каталоги            |
| Shopping Ads    | Товарные пины из каталога            | E-commerce, продажи           |
| Collections Ads | Главное изображение + 3 товара снизу | E-commerce, коллекции товаров |

**Таргетинг:**

- **Демография:** возраст, пол, гео, язык
- **Интересы:** категории (дом и сад, мода, еда и т.д.)
- **Ключевые слова:** по поисковым запросам
- **Аудитории:** ретаргетинг (посетители сайта, вовлечение), Actalike (похожие)
- **Места размещения:** домашняя лента, поиск, похожие пины

**Стратегии кампаний:**

| Цель кампании  | Стратегия       | Использование                      |
| -------------- | --------------- | ---------------------------------- |
| Узнаваемость   | Brand awareness | Охват аудитории, показы            |
| Видеопросмотры | Video views     | Вовлечение через видео             |
| Трафик         | Consideration   | Переходы на сайт                   |
| Конверсии      | Conversions     | Покупки, регистрации, лиды         |
| Каталог        | Catalog sales   | Автоматическое продвижение товаров |

**Бюджет:**

- Минимальный дневной: $10 (≈900₽)
- Рекомендуемый для теста: $20-50/день (≈1800-4500₽)
- Оплата: CPC (за клик) или CPM (за показы)

## 8. Product Pins и Shopping

**Роль в связке:** Продажа (прямая ссылка на товары)

**Технические требования:**

- Pinterest Business аккаунт
- Каталог товаров (data feed)
- Формат фида: CSV, XML, TSV
- Обязательные поля: id, title, description, link, image_link, price, availability

**Пример фида товаров:**

```xml
<item>
  <id>12345</id>
  <title>Деревянная разделочная доска 40×30 см</title>
  <description>Разделочная доска из массива дуба ручной работы.</description>
  <link>https://shop.com/product/12345</link>
  <image_link>https://shop.com/images/board.jpg</image_link>
  <price>2990 RUB</price>
  <availability>in stock</availability>
  <brand>YourBrand</brand>
  <product_type>Kitchen > Cutting Boards</product_type>
</item>
```

**Преимущества Product Pins:**

- Актуальная цена и наличие (синхронизация с сайтом)
- Значок "Shop" (повышает доверие)
- Приоритет в поисковой выдаче
- Checkout (покупка без перехода на сайт — только в США)

## 9. Аналитика Pinterest

**Pinterest Analytics:**

**Основные метрики:**

- **Impressions (Показы):** сколько раз пины были показаны
- **Outbound Clicks (Исходящие клики):** переходы на сайт
- **Saves (Сохранения):** сколько раз пины сохранили в доски
- **Close-ups:** клики для увеличения изображения

**Метрики для отслеживания:**

- **Engagement Rate (ER):** (Saves + Clicks) / Impressions × 100%
  - Хорошо: ≥ 2%
  - Отлично: ≥ 5%
- **CTR (Click-Through Rate):** Clicks / Impressions × 100%
  - Хорошо: ≥ 0.5%
  - Отлично: ≥ 1%
- **Saves:** Высокий показатель = долгосрочный трафик
- **Traffic to site:** Переходы на сайт за месяц

**Отслеживание конверсий:**

- Установка Pinterest Tag на сайт
- Настройка событий (pagevisit, viewcategory, addtocart, checkout)
- Интеграция с Google Analytics (UTM-метки)

## 10. Оптимизация и рост

**Стратегия роста:**

1. **Контент-план на месяц:**

   - Определить 5-10 тем (досок)
   - Создать 50-100 пинов на месяц
   - Распределить по дням (3-5 пинов/день)

2. **Rich Pins (расширенные пины):**

   - Article Rich Pins (статьи): заголовок, автор, описание
   - Product Rich Pins (товары): цена, наличие
   - Recipe Rich Pins (рецепты): ингредиенты, время
   - Требуют мета-теги на сайте

3. **Оптимизация профиля:**

   - Claimed website (подтверждённый сайт)
   - Полное описание профиля с ключевыми словами
   - Ссылка на сайт
   - Регулярная активность

4. **Сезонность:**
   - Планирование контента за 30-45 дней до сезона
   - Пример: Новый год (контент с ноября), Лето (контент с апреля)
   - Использование Pinterest Trends для прогнозов

## 12. Типичные ошибки

**Избегать:**

- ❌ Горизонтальные изображения (плохо видны в ленте)
- ❌ Отсутствие ключевых слов в описаниях
- ❌ Только свой контент (0% репинов)
- ❌ Спам (слишком много пинов за раз)
- ❌ Нерегулярная активность
- ❌ Игнорирование SEO
- ❌ Отсутствие призыва к действию

**Делать:**

- ✅ Вертикальные изображения 2:3
- ✅ SEO-оптимизированные описания
- ✅ Баланс: 80% репинов, 20% свой контент
- ✅ Распределение пинов (3-5/день равномерно)
- ✅ Ежедневная активность
- ✅ Ключевые слова везде
- ✅ CTA в описании ("Узнать больше на сайте")
