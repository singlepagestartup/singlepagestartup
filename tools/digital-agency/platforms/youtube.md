# YouTube

**Общая информация:**

- **Аудитория:** Глобальная, все возрасты, 2+ млрд пользователей
- **Особенности:** SEO-выдача в Google, монетизация, рекомендательный алгоритм
- **Преимущества:** Долгий срок жизни контента, поисковый трафик, связка коротких и длинных видео
- **Статистика:** YouTube Studio (просмотры, удержание, подписки, источники трафика)

---

## 1. YouTube Shorts (короткие вертикальные видео)

**Роль в связке:** Охват → переход на YouTube-канал или внешний ресурс

**Технические требования:**

- Формат: 9:16 (вертикальное)
- Разрешение: 1080×1920 px
- Длительность: 15-60 секунд (оптимально 30-50)
- Субтитры: ОБЯЗАТЕЛЬНЫ
- Музыка: библиотека YouTube или без АП

**Структура ролика:**

| Секунды | Элемент          | Цель               | Примечания                         |
| ------- | ---------------- | ------------------ | ---------------------------------- |
| 0-1     | Супер-крючок     | Остановить скролл  | Максимально сильный визуал + текст |
| 1-5     | Контекст         | Объяснить, о чём   | Быстро, без воды                   |
| 5-25    | Основной контент | Дать ценность      | Пошагово, с визуалом               |
| 25-40   | Доказательство   | Показать результат | Цифры, до/после                    |
| 40-50   | CTA              | Направить дальше   | Подписка / полное видео            |

**Особенности YouTube Shorts:**

- Алгоритм ценит удержание — первые 3 секунды критичны
- Комментарии влияют на продвижение — задавать вопросы
- Loops (петли) — видео начинается с конца предыдущего просмотра
- Связка с длинными видео — «Полная версия на канале»

**Типы контента с высоким охватом:**

1. Разборы ошибок: «3 ошибки, которые убивают ваш [X]»
2. Пошаговые инструкции: «Сделай это за 30 секунд»
3. До/После: трансформации, результаты
4. Реакции: на тренды, новости ниши
5. Мини-туториалы: одно конкретное действие

**CTA варианты:**

- «Подпишись — новый совет каждый день»
- «Полное видео по ссылке в описании»
- «Напиши в комментариях, если хочешь продолжение»
- «Сохрани, чтобы не потерять»

**Метрики успеха:**

- Просмотры: зависит от ниши
- Удержание > 40%
- Подписки с Shorts > 1% от просмотров

## 2. YouTube (длинные видео)

**Роль в связке:** Охват + прогрев → подписка на канал / переход на внешний ресурс

**Технические требования:**

- Формат: 16:9 (горизонтальное)
- Разрешение: 1920×1080 (Full HD) или 4K
- Длительность: 8-20 минут (оптимально 10-15)
- Превью: 1280×720 px, лицо + текст + контраст

**Требования к тексту:**

| Элемент      | Максимум      | Оптимально   | Важно                                                            |
| ------------ | ------------- | ------------ | ---------------------------------------------------------------- |
| **Название** | 100 символов  | 70 символов  | В поиске видно только 60-70 символов                             |
| **Описание** | 5000 символов | 3000-4000    | Первые 150 символов видны без клика "Показать ещё"               |
| **Хештеги**  | 15 хештегов   | 5-7 хештегов | Первые 3 показываются над названием. Если >60 — все игнорируются |
| **Теги**     | 500 символов  | 5-8 тегов    | Общий лимит на все теги                                          |

**Критичные правила:**

- **Первые 150 символов описания** — видны без клика, должны содержать главную информацию
- **Первые 100-157 символов** — критичны для SEO в YouTube
- **Хештеги:** максимум 15, оптимально 5-7, короткие (1-2 слова)
- **Таймкоды (chapters):** автоматически создаются из описания в формате `0:00 - Название`

**Структура видео:**

| Минуты      | Элемент     | Содержание                           | Цель                   |
| ----------- | ----------- | ------------------------------------ | ---------------------- |
| 0:00-0:30   | Крючок      | Обещание результата + интрига        | Удержать первые 30 сек |
| 0:30-2:00   | Введение    | Контекст + представление + структура | Объяснить, о чём видео |
| 2:00-4:00   | Проблема    | Боли ЦА с примерами                  | Создать резонанс       |
| 4:00-12:00  | Решение     | Пошаговый разбор с примерами         | Дать ценность          |
| 12:00-14:00 | Кейс        | Реальный пример применения           | Доказательство         |
| 14:00-15:00 | Итоги + CTA | Резюме + призыв к действию           | Конверсия              |

**Элементы удержания:**

- Pattern interrupts каждые 2-3 минуты (смена кадра, B-roll, графика)
- Open loops: «Об этом расскажу через 2 минуты»
- Визуальные подсказки: стрелки, выделения
- Главы (timestamps) для навигации
- Карточки YouTube с ссылками

**Превью (thumbnail):**

- Лицо с эмоцией (удивление, радость)
- 3-5 слов крупным текстом
- Контрастные цвета
- Читаемость на мобильных (маленький размер)

**CTA в видео:**

- Середина: «Подпишитесь, если полезно»
- Конец: «Ссылка в описании — бесплатный [лидмагнит]»
- Описание: ссылки на ресурсы, Telegram, сайт

**Метрики успеха:**

- Удержание > 50%
- CTR превью > 5%
- Подписки с видео > 2%

## 3. YouTube канал (подписка)

**Роль в связке:** Слежение → глубокий прогрев через видеоконтент → переход к продаже

**Оформление канала:**

- Баннер: 2560×1440 px, с УТП и ссылками
- Аватар: 800×800 px
- Описание канала: о чём канал + ссылки на ресурсы
- Featured channels: дружеские каналы

**Инструменты прогрева:**

- Community tab: опросы, обсуждения, скриншоты
- Плейлисты: тематические серии видео
- Карточки и финальный экран: перелинковка видео
- Shorts: быстрый охват → подписка

**Частота:** 1-2 видео в неделю + Community tab 2-3 раза в неделю

## 4. YouTube Ads (реклама)

**Роль в связке:** Платный охват → подписка на канал / переход на сайт

**Форматы рекламы:**

### 4.1. In-Stream (видеореклама перед/во время видео)

**Технические требования:**

- Формат: 16:9
- Разрешение: 1920×1080
- Длительность: 15-30 секунд (можно длиннее, но пропуск через 5 сек)

**Структура:**

- 0-5 сек: Сильный крючок (до кнопки «Пропустить»)
- 5-15 сек: Проблема + решение
- 15-30 сек: CTA

### 4.2. Discovery (реклама в поиске и рекомендациях)

**Элементы:**

- Превью: 1280×720 px
- Заголовок: до 100 символов
- Описание: до 2 строк

**Настройка:** Через Google Ads → YouTube Ads

## 5. Интеграция форматов YouTube

**Типичные связки внутри экосистемы YouTube:**

### Связка 1: Shorts → Длинные видео

```
YouTube Shorts (охват) → «Полная версия на канале» → Длинное видео (прогрев) → Подписка
```

### Связка 2: Органика + реклама

```
Длинные видео (органика) → YouTube Ads (масштабирование лучших) → Подписка на канал
```

### Связка 3: Канал как воронка

```
Shorts (охват) → Подписка → Длинные видео (прогрев) → CTA в описании → Внешний ресурс
```

### Связка 4: SEO-трафик

```
Поиск YouTube/Google → Длинное видео → Плейлист → Бинж-просмотр → Подписка
```

**Преимущества работы с YouTube:**

- SEO: видео индексируются в Google
- Долгий срок жизни контента (годами получают просмотры)
- Связка Shorts + длинные видео в едином канале
- Монетизация контента (дополнительный доход)
- Community tab для прогрева без создания видео

## 6. Аналитика YouTube

**YouTube Studio:**

- Просмотры и удержание
- Источники трафика (поиск, рекомендации, внешние)
- Подписки и отписки
- Клики по карточкам и конечным заставкам
- Демография аудитории

**Ключевые метрики для отслеживания:**

| Метрика                  | Целевое значение |
| ------------------------ | ---------------- |
| Удержание первых 30 сек  | > 70%            |
| Средний % просмотра      | > 50%            |
| CTR превью               | > 5%             |
| Подписки с видео         | > 2%             |
| Клики по внешним ссылкам | > 1%             |
| Источник: рекомендации   | Рост             |
| Источник: поиск          | Стабильный       |

**Инструменты:**

- YouTube Studio (встроенная аналитика)
- Google Analytics (для переходов на сайт)
- TubeBuddy / VidIQ (расширенная аналитика и SEO)
