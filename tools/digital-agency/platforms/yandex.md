# Яндекс

**Общая информация:**

- **Аудитория:** Рунет, все возрасты, основная поисковая система в России
- **Особенности:** Поисковая реклама + контентная платформа (Дзен)
- **Преимущества:** Высокая конверсия поисковой рекламы, органический охват через Дзен
- **Статистика:** Яндекс Метрика, Яндекс Директ (для рекламы), Дзен Аналитика

---

## 1. Яндекс Дзен (статьи)

**Роль в связке:** Охват через рекомендации → подписка на канал Дзен / внешний ресурс

**Технические требования:**

- Длина: 2000-5000 знаков (оптимально 3000-4000)
- Изображения: минимум 2-3, горизонтальные
- Обложка: 1200×630 px, яркая, с текстом

**Структура статьи:**

```
## Заголовок (50-70 символов)
[Цепляющий, с выгодой или интригой, без кликбейта]

### Введение (200-300 знаков)
[Проблема + обещание решения + для кого статья]

### Основная часть
#### Подзаголовок 1 (проблема или первый шаг)
[Абзац 300-500 знаков + изображение]

#### Подзаголовок 2 (решение или второй шаг)
[Абзац 300-500 знаков + изображение]

#### Подзаголовок 3 (пример или третий шаг)
[Абзац 300-500 знаков + пример/кейс]

### Заключение + CTA (200-300 знаков)
[Резюме + ссылка на лидмагнит / канал Дзен / Telegram]
```

**Особенности Яндекс Дзен:**

- Дочитываемость > 40% — главная метрика
- Алгоритм продвигает статьи с высоким вовлечением
- Ссылки на внешние ресурсы разрешены
- Нельзя кликбейт — пессимизация

**Типы заголовков:**

1. Как + результат: «Как увеличить продажи на 40% за месяц»
2. Число + результат: «5 способов снизить затраты на рекламу»
3. Ошибки: «7 ошибок, из-за которых вы теряете клиентов»
4. Секрет/Метод: «Метод, который используют топ-компании»
5. Личный опыт: «Как я заработал первый миллион на [X]»

**CTA варианты:**

- «Подписывайтесь на канал — завтра продолжение»
- «Полный чек-лист — ссылка в описании канала»
- «Больше таких разборов — в Telegram [ссылка]»

**Метрики успеха:**

- Дочитываемость > 40%
- Переходы по ссылкам > 1%
- Подписки на канал

## 2. Яндекс Директ (платный трафик)

**Роль в связке:** Платный поисковый охват + баннеры → переход на сайт / лендинг

### 2.1. Поисковая реклама

**Роль:** Охват пользователей с явным спросом

**Технические требования:**

| Элемент             | Лимит              | Рекомендация                |
| ------------------- | ------------------ | --------------------------- |
| Заголовок 1         | до 56 символов     | Ключевое слово + выгода     |
| Заголовок 2         | до 30 символов     | Дополнительная выгода / CTA |
| Текст объявления    | до 81 символа      | Детали + призыв             |
| Отображаемая ссылка | до 20 символов     | Релевантный путь            |
| Уточнения           | до 25 символов × 4 | Преимущества списком        |
| Быстрые ссылки      | до 30 символов × 8 | Разделы сайта               |

**Структура объявления:**

```
Заголовок 1: [Ключевое слово] — [Главная выгода]
Заголовок 2: [Дополнительная выгода] или [CTA]

Текст: [Детали предложения]. [Уникальность]. [Призыв к действию]!

Уточнения: ✓ Гарантия | ✓ Бесплатная доставка | ✓ Опыт 10 лет | ✓ 1000+ клиентов

Быстрые ссылки: Каталог | Цены | Отзывы | Контакты | Акции | О компании
```

**Примеры по типам бизнеса:**

**B2B услуги:**

```
Заголовок 1: Внедрение CRM — Рост продаж на 30%
Заголовок 2: Бесплатный аудит за 30 минут

Текст: Настроим amoCRM под ваш бизнес за 2 недели. 150+ внедрений. Оставьте заявку!
```

**E-commerce:**

```
Заголовок 1: Кроссовки Nike Air Max — от 8 990₽
Заголовок 2: Доставка за 1 день по Москве

Текст: Оригинал 100%. Примерка при получении. Возврат 14 дней. Закажите сейчас!
```

**Услуги B2C:**

```
Заголовок 1: Ремонт квартир под ключ — от 5000₽/м²
Заголовок 2: Смета бесплатно за 24 часа

Текст: Фиксированная цена в договоре. Гарантия 3 года. Рассрочка 0%. Звоните!
```

### 2.2. РСЯ (Рекламная сеть Яндекса)

**Роль:** Охват пользователей на сайтах-партнёрах

**Форматы и размеры изображений:**

| Формат             | Размер             | Применение           |
| ------------------ | ------------------ | -------------------- |
| Стандартный баннер | 240×400 px         | Боковые блоки        |
| Горизонтальный     | 728×90 px          | Шапка сайта          |
| Квадратный         | 300×250 px         | Встроенный в контент |
| Широкоформатный    | 1080×607 px (16:9) | Нативная реклама     |
| Смарт-баннеры      | Адаптивные         | Товарные кампании    |

**Требования к изображениям:**

- Без мелкого текста (не более 20% площади)
- Без агрессивных цветов и мигания
- Продукт/услуга в центре внимания
- Читаемость на мобильных устройствах
- Формат: JPG, PNG (до 10 МБ)

**Структура баннера:**

```
┌─────────────────────────────┐
│ [Логотип]                   │
│                             │
│ [ВИЗУАЛ ПРОДУКТА/УСЛУГИ]    │
│                             │
│ [Заголовок — выгода]        │
│ [Подзаголовок — детали]     │
│                             │
│ [CTA-кнопка]                │
└─────────────────────────────┘
```

**Текст для РСЯ (короче, чем поиск):**

```
Заголовок: [Главная выгода] — до 33 символов
Текст: [Детали + CTA] — до 75 символов
```

**Пример:**

```
Заголовок: Английский за 3 месяца
Текст: Онлайн с носителем. Первый урок бесплатно. Запишитесь!
[Изображение: человек с ноутбуком, улыбается]
```

### 2.3. Ретаргетинг

**Роль:** Возврат пользователей, которые взаимодействовали с сайтом

**Сегменты для ретаргетинга:**

| Сегмент                           | Сообщение                                      | CTA              |
| --------------------------------- | ---------------------------------------------- | ---------------- |
| Посетили сайт, не оставили заявку | «Вы смотрели [X]. Получите скидку 10%»         | «Вернуться»      |
| Бросили корзину                   | «Ваши товары ждут! Скидка 15% до конца дня»    | «Оформить заказ» |
| Клиенты (повторная покупка)       | «Специально для вас — новинки со скидкой»      | «Смотреть»       |
| Читали блог                       | «Понравилась статья? Получите бесплатный гайд» | «Скачать»        |

## 3. Интеграция форматов Яндекса

**Типичные связки внутри экосистемы Яндекс:**

### Связка 1: Органика Дзен + Директ

```
Яндекс Дзен (органический охват) → Подписка на канал → Яндекс Директ (масштабирование) → Сайт
```

### Связка 2: Поиск → Ретаргетинг

```
Яндекс Директ Поиск (горячий трафик) → Посещение сайта → Ретаргетинг РСЯ → Конверсия
```

### Связка 3: Холодный трафик → Прогрев

```
РСЯ (баннеры, холодный трафик) → Лендинг + лидмагнит → Email-рассылка → Продажа
```

### Связка 4: Контент + реклама

```
Яндекс Дзен (прогрев статьями) → Look-alike аудитория → Яндекс Директ → Продажа
```

**Преимущества работы с Яндексом:**

- Доминирующая поисковая система в Рунете
- Высокая конверсия поисковой рекламы (явный спрос)
- Яндекс Дзен — бесплатный органический охват
- РСЯ — широкий охват на партнёрских сайтах
- Интеграция с Яндекс Метрикой для точной аналитики

## 4. Аналитика Яндекс

**Яндекс Метрика:**

- Источники трафика
- Поведение на сайте (глубина, время)
- Конверсии и цели
- Карты кликов и скроллинга
- Вебвизор (записи сессий)

**Яндекс Директ:**

- CTR (кликабельность)
- CPC (стоимость клика)
- Конверсии
- ROI / ROAS

**Дзен Аналитика:**

- Просмотры статей
- Дочитываемость
- Подписки на канал
- Переходы по ссылкам

**Ключевые метрики:**

| Метрика               | Целевое значение |
| --------------------- | ---------------- |
| CTR поисковой рекламы | > 8%             |
| CTR РСЯ               | > 0.5%           |
| Дочитываемость Дзен   | > 40%            |
| Конверсия с поиска    | Зависит от ниши  |
| Стоимость конверсии   | В рамках бюджета |

**Инструменты:**

- Яндекс Метрика (аналитика сайта)
- Яндекс Директ (управление рекламой)
- Дзен Аналитика (статистика канала)
