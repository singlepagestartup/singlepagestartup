# Спецификации площадок

```
SYSTEM FILE: platforms/README.md
PURPOSE: База знаний о площадках для всех агентов. Содержит каталог платформ, особенности алгоритмов, проверенные техники.
INPUT: Обновляется на основе опыта работы, исследований и результатов проектов.
OUTPUT: Справочная информация для формирования связок, создания контента и анализа эффективности.
OWNER: Все агенты (чтение), Маркетинговый директор, Архитектор контента, Аналитик (обновление).
GOAL: Предоставить агентам актуальную информацию о площадках для принятия решений. НЕ создаёт артефакты проекта.
```

#### Чек-лист качества рекламного креатива

**Текст:**

- [ ] Крючок в первых словах
- [ ] Конкретные цифры вместо общих фраз
- [ ] Выгоды, а не характеристики
- [ ] Один чёткий CTA
- [ ] Соответствие лимитам платформы

**Изображение:**

- [ ] Продукт/результат в центре
- [ ] Текст не более 20% площади
- [ ] Читаемость на мобильных
- [ ] Контрастные цвета
- [ ] Соответствие размерам платформы

**Видео:**

- [ ] Крючок в первые 3 секунды
- [ ] Субтитры обязательны
- [ ] CTA на экране в конце
- [ ] Нативный стиль для TikTok/Reels
- [ ] Соответствие формату платформы
