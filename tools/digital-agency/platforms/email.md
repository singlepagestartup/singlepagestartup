# Email-рассылка

**Общая информация:**

- **Аудитория:** Универсальная, все сегменты
- **Особенности:** Прямая коммуникация, персонализация, автоматизация
- **Преимущества:** Высокая конверсия, сегментация, отсутствие зависимости от алгоритмов платформ
- **Статистика:** Email-платформы (MailChimp, SendPulse, Unisender), открываемость 15-30%, кликабельность 2-5%

---

## 1. Email-рассылка (канал слежения и прогрева)

**Роль в связке:** Слежение → прогрев → продажа

**Технические требования:**

- Формат: HTML или Plain Text
- Ширина письма: 600-650 px (для корректного отображения на всех устройствах)
- Тема письма: до 50 символов
- Прехедер: до 100 символов
- Длина текста: 300-500 слов (оптимально)

---

## 2. Welcome-цепочка (онбординг)

**Роль:** Знакомство с брендом → прогрев → первая продажа

**Структура Welcome-цепочки (7 писем):**

| Письмо | День | Тема                      | Содержание              |
| ------ | ---- | ------------------------- | ----------------------- |
| 1      | 0    | Доставка лидмагнита       | Лидмагнит + знакомство  |
| 2      | 2    | Польза #1                 | Решение частой проблемы |
| 3      | 4    | Кейс                      | История успеха          |
| 4      | 6    | Ошибка                    | Разбор типичной ошибки  |
| 5      | 8    | Социальное доказательство | Отзывы + мягкий оффер   |
| 6      | 10   | Возражения                | Ответы на сомнения      |
| 7      | 12   | Оффер                     | Предложение + дедлайн   |

## 3. Структура письма

**Формула email:**

```
Тема: [до 50 символов, без спам-слов]
Прехедер: [до 100 символов, дополняет тему]

Привет, {Имя}!

[Основной текст: 300-500 слов]

[CTA-кнопка]

P.S. [Дополнительный крючок]

--
[Подпись]
[Ссылка на отписку]
```

**Элементы письма:**

### 3.1. Тема письма

**Правила:**

- До 50 символов (мобильные экраны)
- Без спам-слов (БЕСПЛАТНО, СРОЧНО, !!!!)
- Персонализация {Имя} или {Город}
- Польза или интрига

**Примеры тем:**

| Тип            | Пример                            |
| -------------- | --------------------------------- |
| Польза         | Как увеличить продажи на 40%      |
| Интрига        | Вы не поверите, что это сработало |
| Персонализация | {Имя}, специально для вас         |
| Социальное     | 1000+ клиентов уже используют это |
| Ограничение    | Осталось 24 часа — не пропустите  |

### 3.2. Прехедер

**Что это:** Текст, который отображается после темы в почтовом клиенте

**Правила:**

- До 100 символов
- Дополняет тему, не дублирует
- Раскрывает интригу или усиливает пользу

**Пример:**

```
Тема: Как увеличить продажи на 40%
Прехедер: Без больших бюджетов на рекламу — проверенная методика
```

### 3.3. Основной текст

**Структура:**

```
Привет, {Имя}!

[Абзац 1: Крючок — обращение к проблеме или обещание]

[Абзац 2-3: Основной контент — польза, кейс, инструкция]

[Абзац 4: CTA — что сделать дальше]

[P.S. Дополнительный крючок или бонус]
```

**Правила написания:**

- Короткие абзацы (2-3 предложения)
- Разговорный стиль, как другу
- Использовать {Имя} в тексте 1-2 раза
- Визуальное форматирование (эмодзи, списки, жирный текст)
- Одна главная мысль на письмо

### 3.4. CTA (призыв к действию)

**Правила:**

- Одна кнопка CTA на письмо (не несколько ссылок)
- Контрастный цвет кнопки
- Текст на кнопке — глагол действия
- Размер кнопки: минимум 44×44 px (для мобильных)

**Примеры CTA:**

| Цель письма       | Текст на кнопке       |
| ----------------- | --------------------- |
| Скачать материал  | Скачать гайд          |
| Записаться        | Записаться на вебинар |
| Купить            | Купить сейчас         |
| Узнать больше     | Узнать подробности    |
| Ответить на опрос | Пройти опрос          |

### 3.5. P.S. (постскриптум)

**Зачем:** Дополнительный крючок — многие читают только тему и P.S.

**Примеры:**

- P.S. Забыл сказать: первые 50 человек получат бонус [X]
- P.S. Если письмо попало в спам — перетащите его в основную папку
- P.S. Вопросы? Ответьте на это письмо — я читаю каждое сообщение

## 4. Типы рассылок

### 4.1. Welcome-серия (онбординг)

**Цель:** Знакомство с брендом + доставка лидмагнита

**Частота:** 7 писем за 12 дней

### 4.2. Регулярная рассылка (newsletter)

**Цель:** Поддержание контакта + прогрев

**Частота:** 1-2 письма в неделю

**Темы:**

- Советы и инструкции
- Кейсы и истории
- Новости индустрии
- Анонсы контента (видео, статьи)

### 4.3. Промо-рассылка

**Цель:** Продажа продукта/услуги

**Частота:** По необходимости (запуски, акции)

**Структура:**

- Проблема
- Решение (продукт)
- Социальное доказательство
- Оффер + ограничение
- CTA

### 4.4. Реактивация (win-back)

**Цель:** Вернуть неактивных подписчиков

**Триггер:** Нет открытий 30-60 дней

**Структура:**

- Признание проблемы: "Мы скучаем по вам"
- Вопрос: "Что-то не так?"
- Бонус за возвращение
- Опцион отписки (честность)

---

## 5. Сегментация подписчиков

**Основные сегменты:**

| Сегмент          | Критерий               | Что отправлять     |
| ---------------- | ---------------------- | ------------------ |
| Новые подписчики | < 14 дней с подписки   | Welcome-серия      |
| Активные         | Открывают > 50% писем  | Регулярный контент |
| Неактивные       | Не открывают > 30 дней | Реактивация        |
| Покупатели       | Совершили покупку      | Upsell, cross-sell |
| Интересанты      | Кликали, но не купили  | Возражения, кейсы  |

---

## 6. Автоматизация (триггерные письма)

**Типы триггеров:**

| Триггер           | Когда отправляется         | Содержание             |
| ----------------- | -------------------------- | ---------------------- |
| Подписка          | Сразу после подписки       | Лидмагнит + Welcome    |
| Брошенная корзина | Товар в корзине > 1 час    | Напоминание + скидка   |
| День рождения     | День рождения подписчика   | Поздравление + бонус   |
| Годовщина покупки | Год после покупки          | Предложение обновления |
| Неоткрытые письма | 3 письма подряд не открыты | Win-back серия         |

## 7. Интеграция Email в связки

**Типичные связки с Email:**

### Связка 1: Лидмагнит → Welcome → Продажа

```
Внешний трафик (реклама/органика) → Лендинг (лидмагнит) → Email Welcome-серия → Продажа
```

### Связка 2: Контент → Рассылка → Прогрев

```
Telegram/YouTube (контент) → Подписка на email → Регулярная рассылка (прогрев) → Промо-рассылка
```

### Связка 3: Вебинар → Догрев → Продажа

```
Вебинар (регистрация) → Email-серия до вебинара → Вебинар → Email-серия после → Продажа
```

### Связка 4: Сегментация → Персонализация

```
Опросы в Email → Сегментация по ответам → Персонализированные серии → Релевантные офферы
```

**Преимущества работы с Email:**

- Прямой канал коммуникации (не зависит от алгоритмов соцсетей)
- Персонализация и сегментация
- Автоматизация воронок
- Высокая конверсия (тёплая аудитория)
- Вы владеете базой (в отличие от соцсетей)

## 8. Аналитика Email

**Ключевые метрики:**

| Метрика                   | Целевое значение | Как улучшить                  |
| ------------------------- | ---------------- | ----------------------------- |
| Open Rate (открываемость) | 15-30%           | Улучшить тему и прехедер      |
| CTR (кликабельность)      | 2-5%             | Сильнее CTA, релевантность    |
| Конверсия                 | Зависит от цели  | Оптимизация оффера            |
| Unsubscribe (отписки)     | < 0.5%           | Не спамить, давать пользу     |
| Bounce Rate (отскоки)     | < 2%             | Чистить базу, валидация email |

**Инструменты:**

- MailChimp (англоязычная аудитория)
- SendPulse (Рунет)
- Unisender (Рунет)
- GetResponse (международная)
- ConvertKit (для создателей контента)

**A/B тесты:**

Тестировать:

- Темы писем
- Время отправки
- CTA (текст и дизайн кнопки)
- Длину письма
- Персонализацию

---

## 9. Технические требования

**Дизайн:**

- Адаптивность (mobile-first)
- Ширина: 600-650 px
- Шрифты: системные (Arial, Helvetica) или веб-безопасные
- Размер шрифта: минимум 14px для основного текста

**Доставляемость:**

- SPF, DKIM, DMARC записи настроены
- Избегать спам-слов (БЕСПЛАТНО, СРОЧНО, !!!)
- Баланс текст/изображения (не только картинки)
- Ссылка отписки обязательна
- Не покупать базы — только opt-in подписчики
