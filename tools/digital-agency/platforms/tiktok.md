# TikTok

**Общая информация:**

- **Аудитория:** Глобальная, молодёжь (16-35 лет), 1+ млрд пользователей
- **Особенности:** Самый мощный алгоритм рекомендаций, трендовая платформа
- **Преимущества:** Вирусный охват, дуэты и стич, низкий барьер входа
- **Статистика:** TikTok Analytics (просмотры, досмотры, вовлечённость, источники)

---

## 1. TikTok Видео (короткие вертикальные видео)

**Роль в связке:** Охват → переход в Telegram / Instagram / сайт

**Технические требования:**

- Формат: 9:16 (вертикальное)
- Разрешение: 1080×1920 px
- Длительность: 15-180 секунд (оптимально 21-34 сек для новых аккаунтов)
- Музыка: трендовые звуки КРИТИЧНЫ для охвата

**Структура ролика:**

| Секунды | Элемент   | Особенности TikTok         |
| ------- | --------- | -------------------------- |
| 0-1     | Крючок    | Максимально провокационный |
| 1-3     | Проблема  | Быстро, эмоционально       |
| 3-15    | Решение   | Пошагово с текстом         |
| 15-21   | Результат | Доказательство             |
| 21-30   | CTA       | Ссылка в bio / комментарии |

**Особенности TikTok:**

- Самый мощный алгоритм рекомендаций
- Дуэты и стич: использовать популярные видео
- Тренды: участвовать в челленджах ниши
- Комментарии: отвечать видео на комментарии
- Green screen: популярный формат для экспертов

**Типы контента:**

1. Storytime: история с эмоциями
2. POV: от первого лица
3. Tutorial: пошаговый разбор
4. Green screen: комментарий к скриншоту/статье
5. До/После: трансформации

**Типы крючков (первые 1 секунда):**

1. Провокационный: «Никто не знает об этом способе...»
2. POV: «POV: ты узнал, что можно зарабатывать на [X]»
3. Шокирующий факт: «Я потерял 1 млн₽, пока не узнал это»
4. Прямое обращение: «Если ты [описание ЦА], смотри до конца»
5. Интрига: «Это изменило мою жизнь»

**CTA варианты:**

- «Ссылка в био — бесплатный гайд»
- «Напиши в комментариях [слово] — пришлю»
- «Подписывайся — новое видео завтра»
- «Часть 2 уже на канале»

**Метрики успеха:**

- Попадание в рекомендации (For You Page)
- Досмотр > 50%
- Репосты > лайков (хороший знак виральности)

## 2. TikTok Ads (реклама)

**Роль в связке:** Платный охват → подписка / переход на сайт

**Форматы рекламы:**

| Формат          | Размер               | Длительность                |
| --------------- | -------------------- | --------------------------- |
| In-Feed Ads     | 9:16 (1080×1920)     | 5-60 сек (оптимально 21-34) |
| Spark Ads       | продвижение органики | —                           |
| TopView         | 9:16                 | до 60 сек                   |
| Branded Hashtag | —                    | —                           |

**Требования:**

- Вертикальный формат ОБЯЗАТЕЛЬНО
- Нативный стиль (как органический контент)
- Музыка: трендовые звуки TikTok
- Субтитры: обязательны
- Без явной «рекламности»

**Структура In-Feed Ads:**

| Секунды | Элемент  | Особенности TikTok                  |
| ------- | -------- | ----------------------------------- |
| 0-1     | Крючок   | Максимально цепляющий, как органика |
| 1-5     | Проблема | Быстро, с юмором или драмой         |
| 5-20    | Решение  | Показать продукт нативно            |
| 20-30   | CTA      | «Ссылка в био» / «Переходи»         |

**Пример сценария In-Feed Ads:**

```
0-1 сек: «POV: ты узнал, что можно зарабатывать на [X]»
1-5 сек: [Реакция удивления] + текст проблемы
5-20 сек: [Показ продукта/процесса в стиле TikTok]
20-30 сек: [Результат + CTA «Ссылка в био»]
```

**Особенности TikTok Ads:**

- Выглядит как обычный TikTok (нативность критична)
- Spark Ads: продвижение уже существующих органических видео (лучшая конверсия)
- TopView: первое видео при открытии приложения (премиум-размещение)
- Branded Hashtag Challenge: вовлечение через челленджи

**Минимальный бюджет:** От $50/день (зависит от региона)

## 3. Интеграция форматов TikTok

**Типичные связки внутри экосистемы TikTok:**

### Связка 1: Органика → Реклама

```
TikTok Органика (тестирование контента) → Spark Ads (масштабирование лучших) → Переход на сайт
```

### Связка 2: Вирусный охват → Внешний канал

```
TikTok Видео (вирусный охват) → CTA в Bio / Комментарии → Telegram / Instagram
```

### Связка 3: Тренды → Аудитория

```
Участие в Трендах (охват) → Дуэты с популярными видео → Подписка → Серия видео (прогрев)
```

### Связка 4: Green Screen → Экспертность

```
Green Screen (комментарий к новостям/скриншотам) → Экспертный контент → Переход в профиль
```

**Преимущества работы с TikTok:**

- Самый мощный алгоритм — новые аккаунты получают охват
- Тренды позволяют быстро набрать аудиторию
- Дуэты и стич — инструменты для увеличения охвата
- Молодая аудитория — высокая виральность
- Spark Ads — продвижение органического контента без создания отдельной рекламы

## 4. Аналитика TikTok

**TikTok Analytics:**

- Просмотры видео
- Досмотры (completion rate)
- Источники трафика (For You, Following, профиль, звук)
- Вовлечённость (лайки, комментарии, репосты)
- Рост подписчиков
- Демография аудитории

**Ключевые метрики:**

| Метрика                  | Целевое значение         |
| ------------------------ | ------------------------ |
| Досмотр видео            | > 50%                    |
| Попадание в For You Page | Да (главная метрика)     |
| Репосты/Лайки            | > 0.3 (знак виральности) |
| Переходы в профиль       | > 2%                     |
| Клики по ссылке в Bio    | > 1%                     |

**Инструменты:**

- TikTok Analytics (встроенная, доступна для Pro аккаунтов)
- TikTok Ads Manager (для рекламных кампаний)
- Сторонние сервисы: Pentos, Analisa.io
