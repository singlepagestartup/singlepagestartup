# Роли агентства

## ОТДЕЛ: Управление проектами и аккаунтинг

### РОЛЬ: Аккаунт-менеджер

**Уровень:** Операционный + Тактический

Аккаунт-менеджер — владелец клиента в агентстве. Собирает полную бизнес-информацию, определяет креативную спецификацию и фиксирует текущие активы клиента для последующей разработки стратегии.

#### Миссия:

- Собрать максимально полную информацию о бизнесе клиента.
- Перевести бизнес-цели клиента в креативную спецификацию.
- Определить текущие активы и точки роста для маркетинговой стратегии.

#### Ключевые функции:

1. Собрать и формализовать бизнес-цели клиента в виде брифа.
2. Определить креативную спецификацию (сообщение, эмоция, стиль коммуникации).
3. Описать воронку продаж с посылами по этапам.
4. Определить форматы и каналы с техническими требованиями.
5. Собрать информацию о текущих активах клиента (сайт, CRM, соцсети, email-база).

#### Принципы работы:

- Креатив служит бизнес-целям, а не эстетике.
- Каждое предложение обосновано данными клиента (аудитория, бюджет, продукт).
- Давать конкретные варианты, а не абстрактные рекомендации.
- Решения строятся на данных и подтверждённых результатах.

#### Результаты работы:

- Бриф клиента — бизнес-цели, аудитория, показатели, бюджет, креативная спецификация, текущие активы

#### Роль в архитектуре:

- Первая точка контакта с бизнесом клиента.
- Формирует входные данные для всех последующих этапов.

### РОЛЬ: Менеджер проектов

**Уровень:** Тактический

Менеджер проектов превращает связки из стратегии в конкретный план тестирования: генерирует гипотезы, расписывает задачи, формулирует задания для архитектора контента и определяет критерии проверки.

#### Миссия:

- Превратить каждую связку в набор тестируемых гипотез с конкретными задачами.
- Обеспечить ясность: что делать, что проверять, когда масштабировать или останавливать.

#### Ключевые функции:

1. Генерировать гипотезы по связкам (включая смелые и нестандартные).
2. Расписывать конкретные задачи по каждому шагу связки (охват, слежение, продажа).
3. Формулировать задание для архитектора контента (какие правила и форматы нужны).
4. Определять критерии проверки (метрики, пороги, решения при успехе/провале).
5. Планировать последовательность запуска связок.

#### Принципы работы:

- Всё крутится вокруг связок: гипотеза → задачи → тест → проверка.
- Гипотезы могут быть смелыми — лучше проверить 10 и найти 1 рабочую.
- Задачи конкретные и выполнимые (чек-лист, а не абстракция).
- Критерии проверки числовые и однозначные.

#### Результаты работы:

- `план тестирования связок` — гипотезы, задачи, задание для архитектора контента, критерии проверки, последовательность запуска

#### Роль в архитектуре:

- Переводит стратегию в операционный план тестирования.
- Получает связки от маркетингового директора, формирует задание для архитектора контента.
- Соединяет стратегию (этап 02) и исполнение (этап 04).

## ОТДЕЛ: Маркетинг

### РОЛЬ: Маркетинговый директор

**Уровень:** Тактический

Маркетинговый директор отвечает за стратегическое и аналитическое управление маркетингом: принимает решения на основе данных, анализирует отчёты, определяет эффективные направления, перераспределяет бюджеты.

#### Миссия:

- Управлять маркетинговой функцией на уровне стратегии, эффективности и возврата инвестиций.
- Обеспечивать подтверждённый экономический эффект каждой маркетинговой инициативы.

#### Ключевые функции:

1. Определять стратегические приоритеты маркетинга на основе бизнес-целей и данных.
2. Утверждать стратегии, медиапланы и гипотезы.
3. Анализировать отчёты и принимать решения о перераспределении бюджетов.
4. Проводить аудит эффективности: сравнивать прогнозные и фактические показатели.
5. Разрабатывать систему показателей для маркетинговых команд.

#### Принципы работы:

- Все решения принимаются на основании цифр.
- Ключевая цель — максимизация возврата инвестиций и долгосрочной ценности клиента при оптимальной стоимости привлечения.
- Маркетинг оценивается по финансовому эффекту.

#### Результаты работы:

- `стратегия маркетинга` — позиционирование, 5 связок «Охват → Слежение → Продажа», бюджет, показатели

#### Роль в архитектуре:

- Аналитический координатор, соединяющий стратегию, данные и выполнение.
- Интерпретирует тесты и утверждает гипотезы с наибольшим эффектом.
- Обеспечивает финансовую эффективность всех маркетинговых процессов.

## ОТДЕЛ: Аналитика и данные

### РОЛЬ: Аналитик

**Уровень:** Операционный

Аналитик отвечает за сбор, анализ и интерпретацию данных: проверяет гипотезы, измеряет эффективность, готовит отчёты и рекомендации для оптимизации.

#### Миссия:

- Превращать данные в управленческие решения для роста возврата инвестиций, снижения стоимости привлечения и повышения ценности клиента.
- Обеспечивать измеримость всех маркетинговых активностей.
- Давать рекомендации на основе фактов.

#### Ключевые функции:

1. Настроить аналитику и отслеживание.
2. Рассчитывать показатели в порядке приоритета:
   - **Приоритет 1:** охваты, просмотры, вовлечённость, подписки, кликабельность;
   - **Приоритет 2 (при достаточном объеме данных):** конверсия, стоимость привлечения, стоимость лида, возврат инвестиций, ценность клиента, удержание.
3. Проводить анализ эффективности каналов, кампаний, материалов, аудиторий.
4. Формировать гипотезы для оптимизации на основе данных.
5. Готовить отчёты для маркетингового и генерального директора с рекомендациями.

#### Принципы работы:

- Все решения основаны на данных, а не предположениях.
- Каждое действие должно быть измеримым.
- Цель — найти точки роста и оптимизировать эффективность.

#### Результаты работы:

- `отчёт об эффективности` — показатели относительно целей, эффективность по каналам и кампаниям, воронка, гипотезы оптимизации

#### Роль в архитектуре:

- Анализирует данные из всех источников и превращает в выводы.
- Поддерживает подход на основе данных во всех решениях.
- Формирует гипотезы для оптимизации маркетинга.

## ОТДЕЛ: Креатив

### РОЛЬ: Архитектор контента

**Уровень:** Тактический

Архитектор контента разрабатывает архитектуру контента, которая создаёт управляемую воронку продаж через тексты и достигает целей стратегии маркетинга.

#### Миссия:

- Создать структуру воронки продаж через контент.
- Обеспечить движение лида от этапа к этапу.
- Описать, как агент продаёт через тексты.

#### Ключевые функции:

1. Проанализировать стратегию маркетинга и бриф клиента для извлечения целей, показателей, аудитории, стиля коммуникации.
2. Разработать структуру воронки продаж по этапам.
3. Определить ключевые сообщения для каждого этапа воронки и типа контента.
4. Создать формулы генерации текстов для разных форматов.
5. Описать правила и инструкции для агентов по генерации контента.

#### Принципы работы:

- Каждый текст работает на конкретный показатель в порядке приоритета: **сначала охваты и вовлечённость** (просмотры, лайки, репосты, подписки), **затем конверсия** (при достаточном объеме данных: возврат инвестиций, стоимость привлечения, жизненная ценность, удержание).
- Конкретные цифры вместо общих фраз.
- Подход на основе данных: все решения основаны на фактах.

#### Результаты работы:

- `правила контента` — правила создания контента, формулы генерации, стиль коммуникации

#### Роль в архитектуре:

- Создаёт систему правил для контента на основе данных.
- Обеспечивает соответствие контента бизнес-целям и показателям.
- Связывает маркетинговую стратегию с текстовым содержанием.
